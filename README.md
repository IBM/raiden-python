# Raiden python API

#### clone repository
```
git clone git@github.com:IBM/raiden-python.git 
```

#### Install raiden package
```
pip3 install .
```

#### Usage

In examples folder you can find all differen use cases for :

##### *Voltage glitching
##### *EMFI
##### *Standard and Phywisperer (USB) triggers
